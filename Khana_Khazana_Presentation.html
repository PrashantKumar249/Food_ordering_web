Online Food Ordering Website - Interactive HTML Presentation

💡 Project Overview

  • Users can order food, manage cart & wishlist
  • Admins manage orders, menu items & generate reports
  • Built with PHP, MySQL, Tailwind CSS, and AJAX

🧱 Technologies Used

  • Frontend: HTML5, Tailwind CSS, AJAX
  • Backend: PHP (Procedural)
  • Database: MySQL

🔐 Authentication & User Features

  • User/Admin login and registration
  • Live search, cart & wishlist
  • Order history and detailed order view
  • Ratings for menu items

🛠️ Admin Features

  • Menu Management (Add/Edit/Delete)
  • View and update orders by status
  • Filterable sales report with summaries

📈 Sales Report Module

  • Shows total orders and revenue
  • Date filtering (from - to)
  • Detailed breakdown of each order

⚙️ Setup Instructions

  1. Copy project to htdocs in XAMPP
  2. Import database SQL file
  3. Edit inc/db.php with DB credentials
  4. Run via localhost/project_folder/

🎯 Conclusion

This project demonstrates a full-stack food ordering system with proper role-based functionality, dynamic UI, and robust backend logic.
